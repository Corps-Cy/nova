嵌入组件

一行代码为您的网站添加 AI 问答

快速接入

将以下代码添加到您的网站 HTML 的 </body> 标签之前：

<!-- Nova AI Widget --> <script> window.novaKbId = "YOUR_KB_ID"; window.novaApiKey = "YOUR_API_KEY"; </script> <script src="http://localhost:3000/widget.js"></script>
配置说明
novaKbId知识库 ID
novaApiKeyAPI Key
widget.js组件脚本地址（指向您的 Nova 服务）
